## 📝 커밋 컨벤션

### ✅ 기본 구조
```bash
type: subject

body (선택)
```

### ✅ type 종류
```
feat: 기능 추가
fix: 버그 수정
refactor: 코드 리팩토링
comment: 주석 추가/수정
docs: 문서 수정
test: 테스트 관련 코드
chore: 빌드/패키지 설정 변경
rename: 파일/폴더명 변경
remove: 파일 삭제
style: 포맷 변경
!BREAKING CHANGE!: 주요 API 변경
```
